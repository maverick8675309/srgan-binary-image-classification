## Super-Resolution GAN (SRGAN) for Binary Image Classification

**Author:** Kristin Kelly

---

## Project Overview

This project investigates whether image super-resolution generated by a Super-Resolution Generative Adversarial Network (SRGAN) can improve the performance of a binary image classifier.

Two transfer-learning models are developed and compared:

- **Classifier A:** Trained on the original Dogs vs. Cats images resized to **128 × 128** pixels.
- **Classifier B:** Trained using high-resolution images generated by an SRGAN.

The performance of both classifiers will be evaluated using multiple metrics, including:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

---

# Objectives

The objectives of this project are to:

- Build a baseline transfer-learning classifier using PyTorch.
- Train an SRGAN to generate high-resolution images from low-resolution inputs.
- Use the SRGAN-generated images to train a second classifier.
- Compare both classifiers using standard evaluation metrics.
- Document a fully reproducible deep learning workflow.

---

# Dataset

This project uses the **Dogs vs. Cats** image dataset.

The original dataset is **not included** in this repository due to its size.

The project uses:

- 22,438 images
- Binary classification (Cat vs. Dog)
- Images resized to **128 × 128**
- Stratified **70% training / 30% testing** split

The reproducible train/test split is included in this repository as:

```
data/manifests/
├── train_70_percent.csv
└── test_30_percent.csv
```

To reproduce the project:

1. Download the Dogs vs. Cats dataset.
2. Extract it into:

```
data/raw/extracted/dogs-vs-cats-classification/
```

The notebooks will automatically use the saved manifest files to reproduce the same dataset split.
