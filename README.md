# srgan-binary-image-classification
# Super-Resolution GAN for Binary Image Classification

**Graduate Midterm Project**  
**Applied Artificial Intelligence & Deep Learning**

---

## Overview

This project investigates whether image super-resolution can improve the performance of transfer learning for binary image classification.

A Super-Resolution Generative Adversarial Network (SRGAN) is trained to reconstruct high-resolution images from low-resolution inputs. The generated images are then used to train a second transfer learning classifier, allowing comparison with a baseline classifier trained on the original high-resolution images.

The project is implemented entirely in **PyTorch** and follows the methodology described in the original SRGAN paper by Ledig et al.

---

## Research Question

Can super-resolution generated images improve binary image classification performance compared to training directly on the original images?

## Objectives

- Train a binary image classifier using transfer learning.
- Develop a Super-Resolution GAN capable of reconstructing 128×128 images from 32×32 inputs.
- Train the SRGAN for at least 150 epochs.
- Generate high-resolution training images using the trained generator.
- Train a second classifier using the generated images.
- Compare classifier performance using multiple evaluation metrics.

## Dataset

Binary image classification dataset consisting of two classes.

Dataset split:

- Training: 70%
- Testing: 30%

Image sizes:

- Original Images: **128 × 128**
- SRGAN Input Images: **32 × 32**
- SRGAN Output Images: **128 × 128**

---

## Data Preprocessing

The following preprocessing steps are applied:

- Image resizing
- Image normalization
- Random horizontal flipping
- Random rotation
- Color jitter
- Tensor conversion

Sample transformed images are displayed within the notebook.

---

## Model A

Baseline transfer learning classifier trained using the original 128×128 images.

Features:

- Transfer Learning
- Frozen convolutional backbone
- Fine-tuned classifier head
- Early stopping
- Model checkpointing

---

## Super-Resolution GAN

The SRGAN consists of:

### Generator

- Residual blocks
- Skip connections
- Upsampling layers
- Convolutional output layer

### Discriminator

- Convolutional neural network
- LeakyReLU activations
- Binary real/fake classification

### Loss Functions

- Content Loss
- Adversarial Loss
- Perceptual Loss

---

## Model B

A second transfer learning classifier trained using images generated by the SRGAN.

The architecture and training procedure mirror Model A to ensure a fair comparison.

---

Both classifiers are evaluated using:
